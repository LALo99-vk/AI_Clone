# AI Automation Bot Project

This project uses [LangChain](https://langchain.com/) and [Ollama](https://ollama.com/) to build a chatbot agent capable of automating tasks such as sending emails, scheduling meetings, assigning team tasks, and basic analytics. 

## Structure
- `src/`: Source code for the agent, tool modules, entry point
- `tests/`: Unit tests for agents and tool functions
- `configs/`: API keys and config files
- `docs/`: Documentation files and usage guides
